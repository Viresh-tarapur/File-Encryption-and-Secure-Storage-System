SecureVault is a Flask + MySQL web application for securely uploading, viewing, downloading, and deleting files with encryption and password protection.

Why We Built This Project

This project was built to learn and demonstrate cybersecurity and full-stack web development skills.
It focuses on secure data storage, user authentication, and encryption techniques to protect files from unauthorized access.

Project Summary

Developed a Python-based secure file system to encrypt and decrypt files using cryptographic techniques, ensuring data confidentiality, secure storage, and protection against unauthorized access.

Features

User Signup & Login

Encrypted File Storage (Fernet)

Upload / View / Download Files

Delete Files with Password Confirmation

Multi-User Support

Auto Logout on Browser Close

Modern Dark UI

Tech Stack

Python, Flask, MySQL, HTML, CSS, Cryptography
